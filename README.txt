Modifies your .htaccess file to redirect all non-whitelisted IP's to static page. Works even if your page is broken.

== Description ==

This plugin is a simple solution for developers, which want to make big changes on their customers sites, and don't want visitors to see broken things while in maintenance mode.

The real Maintenance Mode TMC requires PHP in version 5.3 +

Functionality:
- Single click status toggler on admin bar
- Whitelist
- HTML editor for message
- Simple color picking solution for rapid customization
